# 🎮 Yasir Kasap - Game Developer Portfolio

Unreal Engine ve oyun geliştirme odaklı kişisel portföy web sitesi. **Tamamen HTML, CSS ve Vanilla JavaScript** ile oluşturulmuştur.

![License](https://img.shields.io/badge/license-MIT-blue.svg)

---

## ✨ Özellikler

### 🔐 İnteraktif Mini-Oyunlar
İçeriklere erişmek için çözülmesi gereken 4 farklı mini-oyun:
- **🐛 Bug Hunter** - Hata avı oyunu (blog yazılarını açar)
- **⚡ Wire Fix** - Kablo bağlantı bulmacası
- **🏀 Basket** - Top atma oyunu (Projeler sayfası)
- **🍄 Super Mario Run** - Platform koşu oyunu

### 📄 Sayfalar
- **Ana Sayfa** (`index.html`) - Hero slider, blog yazıları, proje vitrin
- **Projeler** (`projeler.html`) - Unreal Engine ve Unity projeleri
- **Hakkımda** (`hakkimda.html`) - Kariyer yolculuğu ve yetenekler
- **İletişim** (`iletisim.html`) - Formspree entegreli iletişim formu
- **Blog Detay** (`blog-detay.html`) - Tekil blog yazısı şablonu

### 🎨 Tasarım
- Modern, koyu tema (Dark Mode)
- Tamamen responsive (mobil uyumlu)
- CSS animasyonları ve hover efektleri
- Google Fonts (Outfit, Rajdhani)
